GitHub PR Search with Dynamic Dates

Missing required parameters. Please provide "owner", "repo", "tag", and at least one of "created-since-days", "updated-since-days", "merged-since-days", or "closed-since-days".

Parameters

  • owner: The GitHub username or organization that owns the repository. Example: owner=opensearch-project
  • repo: The name of the GitHub repository. Example: repo=OpenSearch
  • tag: The label used to filter PRs. Example: tag=untriaged
  • created-since-days: The number of days back from today to filter PRs by creation date. Example: created-since-days=14
  • updated-since-days: The number of days back from today to filter PRs by last update date. Example: updated-since-days=14
  • merged-since-days: The number of days back from today to filter PRs by merged date. Example: merged-since-days=14
  • closed-since-days: The number of days back from today to filter PRs by closed date. Example: closed-since-days=14

How to Use

To use this redirect page, append the desired options as query parameters in the URL. For example:

  https://peternied.github.io/redirect/pr_search.html?owner=opensearch-project&repo=OpenSearch&merged-since-days=75
